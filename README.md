## Эндпоинты

| Метод | URL | Назначение |
|-------|-----|------------|
| `GET` | `/api/places` | Получить список всех туристических мест |
| `GET` | `/api/places/{id}` | Получить детальную информацию о месте по ID |
| `POST` | `/api/places` | Добавить новое туристическое место |
| `PUT` | `/api/places/{id}` | Обновить данные существующего места |
| `DELETE`| `/api/places/{id}` | Удалить место из системы по ID |

## Коды ответов

| Код | Когда                                                       |
|-----|-------------------------------------------------------------|
| 200 | Успех (GET / PUT)                                           |
| 201 | Создано (POST)                                              |
| 204 | Удалено (DELETE)                                            |
| 400 | Невалидные данные (ошибка валидации, пустые поля)           |
| 404 |  Туристический объект с заданным ID не найден в системе     |

---
## Модель запроса

```json
{
  "name": "Озеро Тургояк",
  "category": "Природа",
  "lat": 55.1542,
  "lon": 60.0611,
  "openTime": "08:00",
  "closeTime": "22:00",
  "avgVisit": 180,
  "description": "Живописное озеро с чистейшей водой"
}
```

| Поле          | Тип     | Ограничения                                  |
|---------------|---------|----------------------------------------------|
| `name`        | string  | Обязательно, уникальное наименование места   |
| `category`    | string  | Обязательно (Природа, Культура, Еда, Отель)  |
| `lat`         | float   | Координата широты объекта, обязательно       |
| `lon`         | float   | Координата долготы объекта, обязательно      |
| `openTime`    | string  | Время открытия в формате ЧЧ:ММ, обязательно  |
| `closeTime`   | string  | Время закрытия в формате ЧЧ:ММ, обязательно  |
| `avgVisit`    | integer | Среднее время визита в минутах, > 0          |
| `description` | string  | Текстовое описание объекта, непустая строка  |
